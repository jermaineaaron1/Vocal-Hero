# Vocal Hero

**Play online:** https://jermaineaaron1.github.io/Vocal-Hero/

A self-contained browser game — no backend, no install. Open `vocal-hero.html` in any modern browser and sing.

## How to play
1. Pick a song from the menu (or create your own).
2. Sing into your microphone and match the pitch of each note as it scrolls by.
3. Score points for accuracy; build streaks for bonus multipliers.

## Song creator
Use the built-in **Create a song** tool to build custom tracks by recording your voice, tapping note timings on a piano roll, and adding lyrics. Songs export as small `.json` files you can share and import.

## Privacy
All microphone processing happens on your device only — no audio is ever sent anywhere.

## Browser support
Requires Web Audio API + `getUserMedia` (Chrome, Firefox, Edge, Safari 14+). Best with headphones in a quiet room.
